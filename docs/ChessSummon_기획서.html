1. 게임 개요

핵심 콘셉트

체스의 이동 규칙을 기반으로 하되, 매 턴 마나를 적립하여 새로운 체스말을 왕 주변에 소환할 수 있는 전략 게임.
초반엔 소수의 폰으로 버티다가, 마나가 쌓이면 강력한 말을 소환해 전세를 역전시키는 흐름이 핵심 재미다.

소환 타이밍의 긴장감

마나를 아껴 퀸을 소환할 것인가, 지금 당장 폰을 소환해 빈틈을 막을 것인가

왕 중심 방어선

소환은 왕 인접에서만 가능하므로, 왕을 지키면서 공격 거점도 구축해야 함

제한된 보드

소환 비용 & 인플레이션

말기본 비용N번째 소환 비용전략 포인트
P폰 1 1 + N 빠른 빈틈 메우기, 물량 공세
N나이트 3 3 + N L자 점프로 기습 가능
B비숍 3 3 + N 대각선 장악, 측면 압박
R룩 5 5 + N 열/행 전체 장악
Q퀸 8 8 + N 가장 강력, 후반 역전카드
같은 종류의 말을 반복 소환할수록 비용이 1씩 증가 (N = 해당 말 소환 횟수). UI 버튼에 현재 실제 비용이 동적으로 표시됨.

소환 규칙

  • 소환 위치: 자신의 왕 기준 인접 8칸 중 빈 칸만 가능
  • 소환하면 이동 불가, 이동하면 소환 불가 (턴당 1회)
  • 소환 직후 해당 말은 그 턴에 이동 불가, 다음 턴부터 이동 가능
  • 왕은 소환 불가 (처음부터 배치)

소환 버튼 조작

  • 소환 버튼 클릭 → 소환 대기 모드 (소환 가능 칸 초록 하이라이트, 버튼 주황 강조)
  • 같은 버튼 재클릭 → 소환 취소 (대기 모드 해제)
  • 다른 버튼 클릭 → 소환 대상 즉시 변경

5. 튜토리얼 시스템

쉬움 난이도 시작 시 선택적으로 진입하는 인게임 튜토리얼. GameScene 위에 TutorialScene 오버레이로 실행되며 6단계 실습을 안내한다.

6단계 미션

단계안내 텍스트완료 조건
1폰을 클릭해보세요말 선택
2이동할 칸을 선택하세요이동 완료
3마나 +2! 소환 버튼을 눌러보세요소환 버튼 클릭
4왕 주변 빈 칸을 클릭해 소환하세요소환 완료
5턴 종료 버튼을 눌러 AI에게 턴을 넘기세요턴 종료
6튜토리얼 완료! 자유롭게 플레이하세요"확인" 버튼 클릭

시각 연출

  • 전체 화면 반투명 오버레이 (depth 20)
  • 현재 단계의 관련 UI 영역에 주황 테두리 점멸
  • 텍스트 박스: 어두운 배경 (#1a1a2e) + 주황 테두리 (#ffaa00)
  • 진행 점(5개): 완료=주황, 현재=흰색, 미완=회색
  • 체크 발생 시 "체크! 이동으로 왕의 위협을 피하세요." 힌트 3초 팝업

6. AI 시스템

난이도배치이동 전략소환 전략
쉬움 자동 랜덤 랜덤 이동 (자기 체크 회피 시도) 랜덤 소환
보통 자동 랜덤 포획 우선 (가치 높은 순, 자기 체크 제외), 이후 랜덤 마나 충족 시 랜덤 소환
어려움 수동 배치 Minimax + 알파베타 가지치기 (탐색 깊이 3) 상황 평가 후 최적 소환

어려움 AI 평가 함수

점수 = Σ(자신의 말 가치) - Σ(상대 말 가치)
     + 중앙(3×3) 장악 보너스 (말당 +0.1)
     + 상대 체크 시 +2
     - 자신 체크 시 -2
     + 상대 체크메이트 시 +10000
5×5 보드 특성상 탐색 깊이 3으로도 충분히 강한 AI 구현 가능

7. 밸런스

폰 중심 초반

마나 0에서 시작하므로 초반 2~3턴은 폰만으로 싸워야 함

왕 노출 리스크

소환하려면 왕 주변 칸이 필요하므로, 왕을 너무 가두면 소환 불가

소환 인플레이션

같은 종류 반복 소환 시 비용 +1 증가 → 단일 말 물량 전략 억제

좁은 보드 후반 강세

5×5에서 룩 1개만 있어도 열 전체를 장악 → 후반 룩/퀸 강세

향후 밸런스 조정 포인트

  • 마나 충전량 조정 (현재 2/턴 → 난이도별 차등 가능)
  • 최대 마나 상한 조정 (현재 10)
  • 소환 비용 개별 조정
  • 소환 인플레이션 폭 조정 (+1 → 다른 값)
  • 왕 주변이 꽉 찼을 때 소환 불가 → 방어가 공격을 막는 딜레마 심화
