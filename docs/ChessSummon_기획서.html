1. 게임 개요

Chess Summon은 5x5 체스판에서 왕을 지키며 마나로 새로운 말을 소환하는 전술 게임이다. 플레이어는 제한된 시야와 60초 턴 안에서 이동 또는 소환을 선택한다.

2. 기본 규칙

  • 보드: 5x5
  • 대전: 플레이어 vs AI
  • 승리 조건: 상대 왕 포획 또는 체크메이트
  • 턴 시작 시 마나 +2, 최대 10
  • 한 턴에 이동 1회, 소환 1회 가능

4. UI / UX

v0.1.7에서는 말을 선택해 이동 가능 칸이 표시된 상태에서도 턴 종료 버튼을 누르면 즉시 턴을 종료할 수 있게 했다. 턴 종료 시 선택된 말, 이동 하이라이트, 소환 대기 상태를 정리해 다음 턴 UI에 이전 선택 상태가 남지 않게 했다.

v0.1.6에서는 보유 마나 표시를 HUD 상단에서 소환 카드 섹션 내부로 이동해 소환 비용과 같은 맥락으로 보이게 했다. 소환 카드의 오른쪽 상태 문구는 제거하고 버튼 색상, 투명도, 비용 아이콘 상태로 선택 가능 여부를 전달해 글자 겹침과 잘림을 줄였다. 소환 안내, 마나 배지, 소환 카드 목록 사이의 Y 좌표를 레이아웃 상수로 분리해 오른쪽 HUD가 서로 침범하지 않도록 했다.

v0.1.5에서는 상단 마나와 소환 비용에 같은 청록색 보석 아이콘을 적용했다. 소환 카드의 비용 텍스트는 제거하고 마나 아이콘 + 숫자로 표시해 비용이 마나임을 직관적으로 알 수 있게 했다.

v0.1.4에서는 말 이미지의 투명 여백을 제거하고 256x256 정규화 에셋으로 재생성했다. 보드 말 렌더 크기는 86px로 조정해 칸 안에서 더 선명하고 크게 보이게 했다. HUD 패널, 소환 카드 행 간격, 하단 버튼 위치도 상수화해 겹침이 재발하지 않도록 했다.

v0.1.3에서는 오른쪽 HUD의 긴 문구를 짧은 공통 문구로 정리하고, 소환 카드 내부 텍스트 크기와 위치를 조정해 글자 겹침과 잘림을 줄였다. 말 렌더 크기는 보드 칸 안에 들어오도록 제한했고, 72x72 public 말 이미지를 1024x1024 원본 캐릭터 에셋으로 교체해 이미지 깨짐을 제거했다.

v0.1.2에서는 게임 화면을 보드 중심 + 오른쪽 턴 운영 패널 구조로 다시 배치했다. 오른쪽 패널에는 이동 1회와 소환 1회 행동 슬롯을 크게 표시해 한 턴에 가능한 행동을 즉시 알 수 있게 했다.

보드 아래에는 상황 힌트가 표시된다. 기본 상태, 말 선택, 소환 카드 선택, 이동 완료, 소환 완료, 행동 완료 상태에 따라 안내 문구가 바뀐다.

? 도움말 버튼에서는 소환 조건, 시야가 밝혀지는 조건, 한 턴 행동 규칙을 언제든 확인할 수 있다. 소환 카드는 말 이름, 아이콘, 비용, 마나 부족 상태를 함께 보여준다.

v0.1.1에서는 어두운 판타지 전술판 스타일로 메뉴, 보드, HUD, 튜토리얼, 모달, 결과 화면을 통일했다. 보드 프레임과 부드러운 안개 타일을 적용하고, 캐릭터 스프라이트 크기와 그림자를 조정해 아트가 더 선명하게 보이도록 했다.
