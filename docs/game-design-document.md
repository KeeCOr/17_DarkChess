## 1. 게임 개요

### 1.1 콘셉트

체스의 이동 규칙을 기반으로 하되, 매 턴 마나를 적립하여 새로운 체스말을 왕 주변에 소환할 수 있는 전략 게임. 초반엔 소수의 폰으로 버티다가, 마나가 쌓이면 강력한 말을 소환해 전세를 역전시키는 흐름이 핵심 재미다.

### 1.2 핵심 재미 요소

- **소환 타이밍의 긴장감** — 마나를 아껴 퀸을 소환할 것인가, 지금 당장 폰을 소환해 빈틈을 막을 것인가
- **왕 중심 방어선** — 소환은 왕 인접에서만 가능하므로, 왕을 지키면서 공격 거점도 구축해야 함
- **제한된 보드** — 5×5라는 좁은 공간에서 순식간에 판세가 뒤집힘
- **60초 턴 제한** — 시간 압박이 더해져 직관적 판단이 중요

### 1.3 플랫폼 로드맵

| 단계 | 플랫폼 | 상태 |
|---|---|---|
| 1단계 | 웹 브라우저 | ✅ 완료 |
| 2단계 | Windows 데스크탑 (Electron) | ✅ 완료 |
| 3단계 | 모바일 (Capacitor) | 예정 |
| 4단계 | Steam | 예정 |

## 2. 게임 규칙

### 2.1 기본 구조

- **보드:** 5×5 격자
- **플레이어:** 2명 (플레이어 vs AI)
  - 플레이어: 하단 (파란색)
  - AI: 상단 (빨간색)
- **승리 조건:** 상대 왕을 잡거나, 체크메이트

### 2.2 시작 배치

**AI (자동 배치)**
```
[ . ][ . ][ K ][ . ][ . ]   ← 행 0: AI 왕 (중앙)
[ P ][ P ][ . ][ P ][ P ]   ← 행 1: AI 폰 4개
```

**플레이어 (배치 페이즈에서 선택)**
- 게임 시작 전 배치 화면에서 하단 2행(행 3~4)에 폰 4개를 자유 배치
- 왕은 하단 중앙(행 4, 열 2)에 고정

### 2.3 턴 순서

1. 턴 시작 → **마나 +2 자동 충전**
2. **이동 또는 소환 중 하나** 선택 (중복 불가)
3. **60초 제한** — 초과 시 자동으로 턴 종료

### 2.4 이동 규칙

표준 체스 이동 규칙을 따름 (단, 캐스링·앙파상 등 특수 규칙 없음).

| 말 | 이동 |
|---|---|
| 폰 (P) | 전방 1칸 이동, 대각선 포획 |
| 나이트 (N) | L자 점프 |
| 비숍 (B) | 대각선 슬라이드 |
| 룩 (R) | 직선 슬라이드 |
| 퀸 (Q) | 비숍 + 룩 |
| 왕 (K) | 인접 1칸 |

> **폰 방향:** 플레이어 폰은 위(행 감소), AI 폰은 아래(행 증가)

### 2.5 체크 & 체크메이트

- **체크:** 왕을 위협하는 말이 빨간 테두리로 강조 표시
- **체크메이트:** 모든 가능한 이동 후에도 체크가 해소되지 않으면 패배
- **왕 직접 포획:** 즉시 패배 처리

## 3. 마나 & 소환 시스템

### 3.1 마나 규칙

| 항목 | 값 |
|---|---|
| 턴당 자동 충전 | +2 |
| 최대 마나 | 10 |
| 시작 마나 | 0 |

### 3.2 소환 규칙

- **소환 위치:** 자신의 왕 기준 인접 8칸 중 **빈 칸**만 가능
- **턴 제한:** 소환하면 이동 불가, 이동하면 소환 불가 (턴당 1회)
- **소환 직후:** 해당 턴에 그 말은 이동 불가
- **왕은 소환 불가** (처음부터 배치)

### 3.3 소환 비용

| 말 | 마나 비용 | 전투 가치 |
|---|---|---|
| 폰 (P) | 1 | 1 |
| 나이트 (N) | 3 | 3 |
| 비숍 (B) | 3 | 3 |
| 룩 (R) | 5 | 5 |
| 퀸 (Q) | 8 | 8 |

> 소환 비용 = 전투 가치 (의도적 설계: 강한 말일수록 오래 기다려야 함)

### 3.4 소환 전략 포인트

- **폰 (1마나):** 2턴이면 소환 가능. 빈틈을 빠르게 메우거나 물량 공세에 유리
- **나이트/비숍 (3마나):** 2턴 적립 필요. 나이트는 L자 점프로 기습 가능
- **룩 (5마나):** 3턴 필요. 열/행을 완전 장악
- **퀸 (8마나):** 4턴 필요. 가장 강력하지만 보드가 이미 정리된 후반에 빛을 발함

## 5. AI 시스템

### 5.1 난이도 구성

| 난이도 | 이동 전략 | 소환 전략 |
|---|---|---|
| **쉬움** | 랜덤 이동 | 랜덤 소환 |
| **보통** | 포획 우선 (가치 높은 순), 이후 랜덤 | 마나 충족 시 랜덤 소환 |
| **어려움** | Minimax + 알파베타 가지치기 (탐색 깊이 3) | 상황 평가 후 최적 소환 |

### 5.2 어려움 AI 평가 함수

```
점수 = Σ(자신의 말 가치) - Σ(상대 말 가치)
     + 중앙(3×3) 장악 보너스 (말당 +0.1)
     + 상대 체크 시 +2
     - 자신 체크 시 -2
     + 상대 체크메이트 시 +10000
```

> 5×5 보드 특성상 탐색 깊이 3으로도 충분히 강한 AI 구현 가능

## 6. 밸런스 고려사항

### 6.1 현재 밸런스 특징

- **폰 중심 초반:** 마나 0에서 시작하므로 초반 2~3턴은 폰만으로 싸워야 함
- **왕 노출 리스크:** 소환하려면 왕 주변 칸이 필요하므로, 왕을 너무 가두면 소환 불가
- **좁은 보드:** 5×5에서 룩 1개만 있어도 열 전체를 장악 → 후반 강세

### 6.2 향후 밸런스 조정 포인트

- 마나 충전량 조정 (현재 2/턴 → 난이도별 차등 가능)
- 최대 마나 상한 조정 (현재 10)
- 소환 비용 개별 조정
- 왕 주변이 꽉 찼을 때 소환 불가 → 방어가 공격을 막는 딜레마 더 심화 가능
